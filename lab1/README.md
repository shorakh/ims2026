# Лабораторная работа №1
## Частотная таблица файла. Энтропия

### Цель работы
Изучить понятие энтропии дискретного источника информации. Научиться строить частотную таблицу символов файла и вычислять энтропию.

### Теоретическая справка
**Энтропия** (по Шеннону) для дискретного источника:

$$H = -\sum_{i=1}^{n} p_i \cdot \log_2(p_i)$$

где $p_i$ — вероятность появления символа $i$, $n$ — мощность алфавита.

**Избыточность:**
$$R = 1 - \frac{H}{H_{max}}, \quad H_{max} = \log_2(n)$$

---

### Задание (общее для всех вариантов)

Написать программу, которая:
1. Читает файл (путь передаётся как аргумент командной строки: `python solution.py input.txt`)
2. Подсчитывает частоту появления каждого **уникального символа**
3. Вычисляет вероятности символов
4. Вычисляет энтропию источника $H$
5. Вычисляет избыточность $R$
6. Выводит результаты в формате, указанном в варианте
