## 1. Цель работы

Изучить понятие энтропии дискретного источника информации. Научиться строить частотную таблицу символов файла и вычислять энтропию по формуле Шеннона.

---

## 2. Краткое описание алгоритма

1. Считываем файл в строку.
2. Подсчитываем, сколько раз встречается каждый символ (`Counter`).
3. Вычисляем вероятность каждого символа: `p_i = count_i / N`.
4. Вычисляем энтропию: `H = -sum(p_i * log2(p_i))`.
5. Выводим топ-5 символов по убыванию частоты (вариант 1).

---

## 3. Описание программы

- `build_frequency_table(text)` — возвращает `Counter` с частотами символов.
- `compute_entropy(freq_table, total)` — вычисляет энтропию H.
- `main()` — читает файл, вызывает функции, выводит результат.
