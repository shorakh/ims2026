# Лабораторная работа №2
## Кодовые таблицы для сжатия файла

### Цель работы
Изучить методы статистического кодирования: код Хаффмана и код Шеннона-Фано. Построить кодовые таблицы и оценить степень сжатия.

### Теоретическая справка

**Код Хаффмана** — оптимальный префиксный код, строится снизу вверх через двоичное дерево.  
**Код Шеннона-Фано** — субоптимальный префиксный код, строится сверху вниз.

**Средняя длина кодового слова:**
$$\bar{L} = \sum_{i=1}^{n} p_i \cdot l_i$$

**Коэффициент сжатия:**
$$K = \frac{L_{исх}}{L_{сж}}, \quad \text{где } L_{исх} = 8 \text{ бит/символ (ASCII)}$$

---

### Задание (общее для всех вариантов)

Написать программу, которая:
1. Читает входной файл (`python solution.py input.txt`)
2. Строит частотную таблицу символов
3. Строит кодовое дерево и таблицу кодов (метод — по варианту)
4. Вычисляет среднюю длину кодового слова $\bar{L}$
5. Вычисляет коэффициент сжатия $K$
6. Выводит таблицу кодов и итоговые метрики
