# Лабораторная работа №3
## Архивация (сжатие) и разархивация файлов

### Цель работы
Изучить методы сжатия данных без потерь. Реализовать один из алгоритмов сжатия, сжать файл и восстановить его из архива.

### Теоретическая справка

**RLE (Run-Length Encoding)** — кодирование серий одинаковых символов.  
**LZ77 / LZ78** — словарные методы сжатия (основа ZIP, gzip).  
**Deflate** — комбинация LZ77 + Хаффман (используется в ZIP, PNG).

**Степень сжатия:**
$$CR = \frac{\text{Размер оригинала}}{\text{Размер архива}}$$

---

### Задание (общее для всех вариантов)

Написать программу с двумя режимами:
```bash
python solution.py compress   input.txt   output.arc
python solution.py decompress output.arc  restored.txt
```

1. **compress**: читает входной файл, сжимает, сохраняет архив
2. **decompress**: читает архив, восстанавливает исходный файл
3. После разархивации проверить, что `restored.txt == input.txt` (побайтово)
4. Вывести степень сжатия
