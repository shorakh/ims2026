# Лабораторная работа №4
## Помехозащитное кодирование

### Цель работы
Изучить методы помехозащитного кодирования. Реализовать кодирование и декодирование с обнаружением и/или исправлением ошибок.

### Теоретическая справка

**Код с чётностью (Parity bit)** — добавляет 1 бит, обнаруживает 1 ошибку.  
**Код Хэмминга** — обнаруживает 2 ошибки, исправляет 1 ошибку.  
**CRC (Cyclic Redundancy Check)** — обнаруживает пакеты ошибок.

**Кодовое расстояние Хэмминга** между словами $a$ и $b$:
$$d(a, b) = \text{число позиций, где } a_i \neq b_i$$

---

### Задание (общее для всех вариантов)

Написать программу, которая:
1. Принимает двоичную строку или байты (из файла или командной строки)
2. Кодирует методом по варианту
3. Вносит **искусственную ошибку** (по номеру бита или случайно)
4. Декодирует, обнаруживает/исправляет ошибку
5. Выводит результат

```bash
python solution.py encode "1011010"
python solution.py decode "10110100110" --error 3
```
