# Лабораторная работа №5
## Шифрование файлов

### Цель работы
Изучить основные криптографические алгоритмы. Реализовать шифрование текстового файла и оценить стойкость шифра.

### Теоретическая справка

**Симметричное шифрование** — один ключ для шифрования и дешифрования.  
**Шифр Цезаря** — сдвиг символов алфавита на константу $k$.  
**Шифр Виженера** — полиалфавитный шифр с ключевым словом.  
**XOR-шифр** — побайтовое исключающее ИЛИ с ключом.  
**Перестановочный шифр** — перестановка символов по ключу.

---

### Задание (общее для всех вариантов)

Написать программу с двумя режимами:
```bash
python solution.py encrypt input.txt encrypted.txt KEY
python solution.py decrypt encrypted.txt decrypted.txt KEY
```

После дешифрования проверить, что `decrypted.txt == input.txt`.
