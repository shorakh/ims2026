# Лабораторная работа №6
## Дешифровка файлов

### Цель работы
Изучить методы криптоанализа. Реализовать атаки на шифры без знания ключа и оценить стойкость алгоритмов шифрования.

### Теоретическая справка

**Частотный анализ** — атака на основе частоты символов в естественном языке.  
**Атака грубой силой (brute-force)** — перебор всех возможных ключей.  
**Метод Касиски** — нахождение длины ключа Виженера.  
**Индекс совпадений** — метрика для оценки «случайности» текста.

**Индекс совпадений:**
$$IC = \frac{\sum_{i=0}^{25} f_i(f_i - 1)}{N(N-1)}$$

где $f_i$ — частота буквы $i$, $N$ — длина текста.  
Для русского языка $IC \approx 0.055$, для случайного — $IC \approx 0.038$.

### Задание (общее для всех вариантов)

Программа получает **зашифрованный файл** и должна попытаться его дешифровать **без знания ключа**.

```bash
python solution.py encrypted.txt
```

Вывести: предполагаемый ключ, дешифрованный текст, уровень уверенности.
