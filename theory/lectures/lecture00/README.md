## 1. Вещественные, энергетические и информационные процессы

В современном обществе можно выделить три типа процессов взаимодействия:

| Тип процесса | Что переносится | Примеры |
|-------------|----------------|---------|
| **Вещественный** | Материя, вещество | Транспорт, производство |
| **Энергетический** | Энергия | Электросети, отопление |
| **Информационный** | Информация | Интернет, телефония, СМИ |

**Информация** — это сведения об объектах и явлениях окружающего мира, их свойствах, состоянии и взаимодействии, уменьшающие имеющуюся неопределённость.

Ключевые свойства информации:
- **Актуальность** — соответствие текущему моменту времени
- **Достоверность** — соответствие реальному состоянию объекта
- **Полнота** — достаточность для принятия решений
- **Доступность** — возможность получения в нужный момент
- **Релевантность** — соответствие решаемой задаче

## 2. Понятие информационной системы

**Информационная система (ИС)** — это взаимосвязанная совокупность средств, методов и персонала, используемых для хранения, обработки и выдачи информации в интересах достижения поставленной цели.

### Структура ИС

```
┌─────────────────────────────────────────────────────┐
│               ИНФОРМАЦИОННАЯ СИСТЕМА                 │
│                                                     │
│  ┌──────────┐   ┌──────────┐   ┌──────────────┐    │
│  │  Ввод    │──▶│Обработка │──▶│    Вывод     │    │
│  │данных    │   │данных    │   │информации    │    │
│  └──────────┘   └──────────┘   └──────────────┘    │
│        ▲               │                            │
│        │        ┌──────▼─────┐                     │
│        │        │  Хранение  │                     │
│        └────────│  данных    │                     │
│                 └────────────┘                     │
└─────────────────────────────────────────────────────┘
```

### Компоненты ИС

1. **Информационная база** — данные и знания
2. **Техническое обеспечение** — компьютеры, сети, периферия
3. **Программное обеспечение** — ОС, СУБД, прикладные программы
4. **Математическое обеспечение** — алгоритмы, модели, методы
5. **Организационное обеспечение** — регламенты, инструкции
6. **Правовое обеспечение** — нормативные документы
7. **Кадровое обеспечение** — персонал

## 3. Основные виды информационных систем

### По масштабу
| Класс | Описание | Примеры |
|-------|----------|---------|
| Персональные | Для одного пользователя | MS Office, 1С |
| Корпоративные | Для предприятия | ERP, CRM |
| Государственные | Для госструктур | ГАС «Выборы», ГИС ЖКХ |
| Глобальные | Международные | Интернет, SWIFT |

### По функциям
- **Транзакционные (OLTP)** — оперативная обработка транзакций (банки, магазины)
- **Аналитические (OLAP)** — анализ и поддержка принятия решений
- **Управленческие (MIS)** — поддержка управленческих решений
- **Экспертные системы** — имитация экспертных знаний

### По архитектуре
- **Централизованные** — один сервер, все данные в одном месте
- **Распределённые** — данные и вычисления на разных узлах
- **Клиент-серверные** — разделение функций клиента и сервера

## 4. Информационное общество

**Информационное общество** — общество, в котором большинство работников заняты производством, хранением, переработкой и реализацией информации.

Признаки информационного общества:
- Более 50% ВВП создаётся в информационной сфере
- Большинство работников — «работники знания»
- Информация является основным товаром
- Развита глобальная информационная инфраструктура
