## 1. Количество информации

### Неопределённость и информация

Информация устраняет **неопределённость**. Чем больше неопределённость была до получения сообщения, тем больше информации несёт это сообщение.

**Аксиомы меры количества информации:**
1. Количество информации — монотонно убывающая функция вероятности
2. Информация от независимых событий складывается
3. Функция непрерывна

Из аксиом следует единственная функция:

$$I(p) = \log_b \frac{1}{p} = -\log_b p$$

При основании $b = 2$ единица измерения — **бит** (binary digit).

---

## 2. Мера Хартли

Для **равновероятных** исходов (вероятность каждого $p = 1/N$):

$$\boxed{I = \log_2 N}$$

где $N$ — число возможных равновероятных исходов.

### Примеры

| Событие | N | I, бит |
|---------|---|--------|
| Подбросить монету | 2 | 1 |
| Бросить кубик | 6 | 2.585 |
| Один символ ASCII | 256 | 8 |
| Один символ из алфавита (33 буквы) | 33 | 5.044 |
| Карта в колоде | 52 | 5.700 |

### Пример задачи

> В ящике 32 одинаковых шара. Сколько информации несёт сообщение о том, какой шар вытащили?

$$I = \log_2 32 = \log_2 2^5 = 5 \text{ бит}$$

---

## 3. Количество информации как случайная величина

Если события **не равновероятны**, количество информации от события $x_i$ с вероятностью $p_i$:

$$I_i = -\log_2 p_i \quad \text{(бит)}$$

Это случайная величина — разные события несут разное количество информации.

| Событие | $p_i$ | $I_i$ (бит) |
|---------|-------|------------|
| Редкое ($p = 0.01$) | 0.01 | 6.644 |
| Среднее ($p = 0.25$) | 0.25 | 2.000 |
| Частое ($p = 0.90$) | 0.90 | 0.152 |

> **Вывод:** чем реже событие, тем больше информации оно несёт при наступлении.

## 4. Энтропия

**Энтропия** — математическое ожидание количества информации, то есть среднее количество информации на одно сообщение источника:

$$\boxed{H(X) = -\sum_{i=1}^{n} p_i \log_2 p_i \quad \text{[бит]}}$$

где $p_i$ — вероятность $i$-го символа, $n$ — мощность алфавита.

По соглашению: $0 \cdot \log_2 0 = 0$.

### Пример вычисления

Источник имеет 3 символа: A (p=0.5), B (p=0.25), C (p=0.25)

$$H = -0.5 \log_2 0.5 - 0.25 \log_2 0.25 - 0.25 \log_2 0.25$$
$$H = -0.5 \cdot (-1) - 0.25 \cdot (-2) - 0.25 \cdot (-2)$$
$$H = 0.5 + 0.5 + 0.5 = 1.5 \text{ бит}$$

---

## 5. Основные свойства энтропии

### Свойство 1: Неотрицательность
$$H(X) \geq 0$$
Равенство нулю только если один из исходов достоверен ($p_k = 1$).

### Свойство 2: Максимум при равномерном распределении
$$H(X) \leq \log_2 n = H_{\max}$$

Максимум достигается при $p_1 = p_2 = \ldots = p_n = \frac{1}{n}$.

```
H
│     ×──── Hmax = log₂n
│   ×   ×
│  ×     ×
│ ×       ×
│×         ×
└──────────────── p
0    равномерное   1
     распределение
```

### Свойство 3: Симметричность
$$H(p_1, p_2, \ldots, p_n) = H(p_{\sigma(1)}, p_{\sigma(2)}, \ldots, p_{\sigma(n)})$$

Перестановка вероятностей не меняет энтропию.

### Свойство 4: Аддитивность для независимых источников
$$H(X, Y) = H(X) + H(Y) \quad \text{если } X \perp Y$$

---

## 6. Избыточность источника

**Максимальная энтропия** (при равномерном распределении):
$$H_{\max} = \log_2 n$$

**Относительная энтропия:**
$$\eta = \frac{H}{H_{\max}}$$

**Избыточность:**
$$\boxed{R = 1 - \eta = 1 - \frac{H}{H_{\max}}}$$

Избыточность показывает, насколько источник «неэффективен». Избыточность русского языка ≈ 60–70%.

---

## 7. Среднее взаимное количество информации

Для двух случайных величин $X$ и $Y$:

$$I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$

$$I(X; Y) = H(X) + H(Y) - H(X, Y)$$

Диаграмма соотношений:

```
     ┌────────────────────────┐
     │   H(X)                 │
     │         ┌──────────────────────┐
     │  H(X|Y) │  I(X;Y)  │  H(Y|X) │
     │         │           │         │
     └─────────┼───────────┤   H(Y)  │
               └──────────────────────┘
```

---

## 8. Непрерывные источники

Для непрерывного источника с плотностью распределения $p(x)$:

**Дифференциальная энтропия:**
$$h(X) = -\int_{-\infty}^{\infty} p(x) \log_2 p(x) \, dx$$

Для **нормального распределения** $\mathcal{N}(\mu, \sigma^2)$:
$$h(X) = \frac{1}{2} \log_2(2\pi e \sigma^2) \text{ бит}$$

## Контрольные вопросы

1. Что такое мера Хартли? При каком условии она применяется?
2. Запишите формулу энтропии Шеннона и объясните каждый символ.
3. При каком распределении вероятностей энтропия максимальна?
4. Что такое избыточность источника? Как она связана с энтропией?
5. Вычислите энтропию источника с символами: A (p=0.6), B (p=0.3), C (p=0.1).

### Задача для самостоятельного решения

> Русский текст содержит 33 буквы. Если бы они были равновероятны, энтропия была бы $H_{\max} = \log_2 33 \approx 5.04$ бит. Реально $H \approx 3.52$ бит. Найдите избыточность русского языка.

<details>
<summary>Решение</summary>

$$R = 1 - \frac{H}{H_{\max}} = 1 - \frac{3.52}{5.04} \approx 1 - 0.698 = 0.302 \approx 30\%$$

Это минимальная оценка; с учётом связей между буквами реальная избыточность выше — около 60–70%.
</details>
