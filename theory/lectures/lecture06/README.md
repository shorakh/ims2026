# Лекция 6. Восприятие информации

> **Раздел 2:** Тема 2.4

---

## 1. Процесс восприятия информации

**Восприятие информации** — активный психофизиологический процесс получения, обработки и интерпретации сигналов из внешней среды.

### Основные этапы восприятия

```
Сигнал → Обнаружение → Различение → Идентификация → Распознавание → Интерпретация
```

1. **Обнаружение** — фиксация наличия сигнала (да/нет)
2. **Различение** — разделение похожих сигналов
3. **Идентификация** — соотнесение с известным образом
4. **Распознавание** — определение принадлежности к классу
5. **Интерпретация** — придание смысла

---

## 2. Аспекты восприятия информации

| Аспект | Описание | Пример |
|--------|----------|--------|
| **Физический** | Параметры сигнала | Длина волны, амплитуда |
| **Морфологический** | Форма и структура | Буква, форма объекта |
| **Синтаксический** | Правила построения | Грамматика, синтаксис |
| **Семантический** | Смысл и значение | Понимание предложения |
| **Прагматический** | Ценность для получателя | Полезность информации |

---

## 3. Пропускная способность человека

Человек воспринимает информацию через органы чувств:

| Канал | Скорость восприятия |
|-------|-------------------|
| Зрение | до 10⁷ бит/с |
| Слух | до 10⁴ бит/с |
| Осязание | до 10² бит/с |
| Сознательная обработка | ~50 бит/с |

**«Бутылочное горлышко»**: сознание обрабатывает значительно меньше, чем воспринимают органы чувств.

---

## 4. Семантические аспекты информации

**Семантика** — смысловое содержание информации.

Проблема: одни и те же данные могут иметь разный смысл для разных получателей.

**Тезаурус** — словарь понятий с семантическими связями. Позволяет формализовать знания.
