## 1. Понятие информационной системы

**Информационная система (ИС)** — совокупность методов и средств, реализующих хранение, обработку и передачу информации для достижения цели.

### Свойства системы

| Свойство | Описание |
|----------|----------|
| **Целостность** | Система не сводится к сумме элементов |
| **Эмерджентность** | Системное качество — сверх суммы частей |
| **Иерархичность** | Система состоит из подсистем |
| **Адаптивность** | Реакция на изменения среды |
| **Устойчивость** | Сохранение функций при возмущениях |

### Структура ИС

- **Функциональная структура** — что делает система (задачи)
- **Информационная структура** — потоки и хранилища данных
- **Техническая архитектура** — аппаратное и программное обеспечение
- **Организационная структура** — пользователи и роли

### Жизненный цикл ИС

```
Анализ → Проектирование → Разработка → Тестирование → Внедрение → Сопровождение
   ↑_______________________________________________|
                    (итерации)
```

## 2. Качество информационной системы

### Стандарты ISO серии 9000

**ISO 9000** — серия международных стандартов системы менеджмента качества:
- **ISO 9001** — требования к системе качества
- **ISO 9004** — рекомендации по улучшению
- **ISO 9126** — качество программного обеспечения

### Характеристики качества ПО (ISO 9126)

| Характеристика | Описание |
|---------------|----------|
| **Функциональность** | Наличие нужных функций |
| **Надёжность** | Устойчивость к отказам |
| **Удобство** | Простота использования |
| **Эффективность** | Использование ресурсов |
| **Сопровождаемость** | Лёгкость изменений |
| **Переносимость** | Работа в разных средах |

### Модель «уровней зрелости» CMM (Capability Maturity Model)

Оценивает зрелость процессов разработки организации:

| Уровень | Название | Описание |
|---------|----------|----------|
| 1 | **Начальный** | Хаотичные процессы |
| 2 | **Повторяемый** | Базовое управление проектами |
| 3 | **Определённый** | Документированные процессы |
| 4 | **Управляемый** | Количественное управление |
| 5 | **Оптимизирующий** | Непрерывное улучшение |
