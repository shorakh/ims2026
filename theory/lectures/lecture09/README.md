## 1. Стандарты в области информационных технологий

**Стандарт** — документ, устанавливающий требования, спецификации, руководящие принципы или характеристики для обеспечения совместимости.

### Уровни стандартизации

| Уровень | Организация | Примеры |
|---------|------------|---------|
| Международный | ISO, IEC, ITU | ISO/IEC 27001, ITU-T |
| Региональный | ETSI, CEN | Стандарты ЕС |
| Национальный | ГОСТ, DIN, ANSI | ГОСТ Р 34.12-2015 |
| Отраслевой | IEEE, W3C, IETF | IEEE 802.11 (Wi-Fi) |
| Корпоративный | Компании | Microsoft NTFS |

### Роль стандартов в ИТ

- **Совместимость** — разные системы работают вместе
- **Конкуренция** — открытые стандарты снижают монополию
- **Инновации** — единая основа для развития
- **Безопасность** — стандарты защиты данных

### Виды стандартов

| Вид | Описание |
|-----|----------|
| **De jure** | Принятый официальной организацией |
| **De facto** | Сложился исторически (PDF, MP3) |
| **Открытый** | Доступен всем без ограничений |
| **Проприетарный** | Принадлежит компании |

## 3. Методология современных информационных систем

### ERP-системы (Enterprise Resource Planning)

**ERP** — интегрированная система управления всеми ресурсами предприятия.

Модули ERP:
```
┌──────────────────────────────────────────────┐
│                 ERP-система                  │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐        │
│  │Финан.│ │  HR  │ │Произв│ │Логис.│  ...   │
│  └──────┘ └──────┘ └──────┘ └──────┘        │
│               Единая база данных             │
└──────────────────────────────────────────────┘
```

Крупнейшие ERP: SAP, Oracle E-Business Suite, Microsoft Dynamics, 1С:ERP.

### CRM-системы (Customer Relationship Management)

Управление отношениями с клиентами: продажи, маркетинг, поддержка.

Примеры: Salesforce, Bitrix24, amoCRM.

### Корпоративные порталы

Единая точка доступа к корпоративным ресурсам и приложениям через веб-браузер.

## 4. CALS-технологии

**CALS** (Continuous Acquisition and Life Cycle Support) — концепция непрерывной информационной поддержки жизненного цикла изделия.

Этапы жизненного цикла изделия:
```
Проектирование → Производство → Эксплуатация → Утилизация
        ↑_______________________________________________|
              Единое информационное пространство
```

**Ключевые принципы CALS:**
- Безбумажный обмен данными
- Параллельное проектирование
- Интеграция данных на всех этапах

## 5. Классификация информационных систем

### По сфере применения

| Класс | Описание | Примеры |
|-------|----------|---------|
| **Производственные** | Управление производством | ERP, MES, SCADA |
| **Коммерческие** | Торговля и логистика | CRM, WMS |
| **Финансовые** | Банки, страхование | АБС, биржевые системы |
| **Транспортные** | Перевозки | Система бронирования |
| **Научные** | Исследования | ГИС, системы моделирования |
| **Образовательные** | Обучение | LMS, электронные университеты |
| **Государственные** | Госуправление | Госуслуги, ГАС |

### По режиму работы

- **Пакетный** — накопление + обработка пакетом (ночная выгрузка)
- **Реального времени** — немедленная реакция (биржа, управление АЭС)
- **Интерактивный** — диалог с пользователем (банкомат, веб-приложение)
