## Задачи

### Задачи по энтропии

**Задача 1.**  
Источник генерирует символы {A, B, C, D} с вероятностями 0.5, 0.25, 0.125, 0.125.  
а) Найдите энтропию H.  
б) Найдите максимальную энтропию Hmax.  
в) Найдите избыточность R.

**Задача 2.**  
В ящике 16 красных и 16 синих шаров. Сколько информации несёт сообщение о цвете вытащенного шара?

**Задача 3.**  
Монету бросают 3 раза. Сколько информации содержит результат? Сравните с $\log_2 8$.

**Задача 4.**  
Телефонная книга содержит 1024 имени. Найдите количество информации для однозначного указания нужного имени.

### Задачи по каналам связи

**Задача 5.**  
Канал имеет полосу 4 кГц и SNR = 30 дБ. Найдите пропускную способность.

**Задача 6.**  
Аудиосигнал имеет полосу 20 кГц. С какой минимальной частотой его нужно дискретизировать? При 16 бит на выборку найдите битрейт.

**Задача 7.**  
При какой вероятности ошибки бита пропускная способность двоичного симметричного канала равна 0?

### Задачи по кодированию

**Задача 8.**  
Постройте код Хаффмана для символов: A(0.4), B(0.3), C(0.2), D(0.1).  
Вычислите: среднюю длину $\bar{L}$, энтропию $H$, КПД $\eta$.

**Задача 9.**  
Для тех же символов постройте код Шеннона-Фано и сравните с Хаффманом.

**Задача 10.**  
Файл размером 80 000 байт содержит 4 символа с частотами: A=40000, B=20000, C=12000, D=8000. Постройте код Хаффмана и найдите размер сжатого файла.

**Задача 11.**  
Закодируйте последовательность `1101` кодом Хэмминга (7,4). Какой бит является контрольным на позиции 2?

**Задача 12.**  
Принята последовательность `1001010`. Синдром = `011`. Найдите позицию ошибки и восстановите исходное слово.

### Задачи по криптографии

**Задача 13.**  
Зашифруйте слово «ПАУК» шифром Цезаря со сдвигом 5 (русский алфавит). Расшифруйте обратно.

**Задача 14.**  
Подсчитайте индекс совпадений для текста «АААААБББББ» (5 букв А, 5 букв Б). Сравните с $IC$ русского языка.
